## 📊 项目背景

### Vibe Coding 是什么?
- **项目名称**: Vibe Coding (AI算力交易平台)
- **核心价值**: 国内全覆盖,不依赖海外 GPU
- **技术栈**: Go Gateway + vLLM + OpenMeter + Stripe
- **团队规模**: 4-8人
- **MVP 工期**: 6-8周

### 为什么记录这些?
1. **复用性**: 下次类似项目,可以快速应用经验
2. **培训**: 新成员快速了解项目决策过程
3. **复盘**: 项目结束后,总结成功和失败点
4. **分享**: 将经验传递给其他项目
